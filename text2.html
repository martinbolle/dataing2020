Tags are used to format the text. Like "Heading 1" and "Paragraph" that I have uses here

Basic tags

Headings -- written h1, h2,.... h5 -- should be familiar from any word processing software.

Not all have closing tags. For example <br> wich insert a linebreak. Then, we usually end it with a forward slash in the end

There are several more formating tags. For example emphasis/italics and strong/bold/fat text.

The most importante principle is consistency. You must use tags cosistently and ensure for esample, that you never follow a H! wit a H3 withouth a H2 inbetween.

Tags that defines text and layout

TThen there are tags to make text clearly separate from the rest. For example we use <code> </code> when we need a wrapper for code/script/tags that should not be executed but displayed as code

We have tags for ordered and unordered lists (<ol> and <ul>)
